Pixel Perfect plugin makes it easier to develop Wordpress sites. Place image overlay over the site which helps with positioning object on the page.

== Description ==

Pixel Perfect plugin makes it easier to develop Wordpress sites. Use it to overlay the page you are working on with design image to achieve pixel perfect layout when coding. You can specify opacity of the overlay, height of the overlay, which page to display it on and activate/deactivate it keeping all your settings. You can also turn it on/off when developing the site by clicking on the bulb in the right top corner.

== Installation ==

1. Download Pixel Perfect plugin zip, unzip it and place it to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to settings and find Pixel Perfect plugin. Upload your image and change your settings. Save the options.
